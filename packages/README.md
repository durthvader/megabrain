# Pacotes compartilhados

Esta pasta contém somente código reutilizado por mais de um projeto. Ela não é
um projeto e não aparece no catálogo. As subpastas são locais e ignoradas pelo
repositório central público.

- cada subpasta representa uma família de componentes compartilhados;
- o nome da família deve corresponder aos sandboxes que a utilizam.

Excluir um sandbox não deve apagar estes pacotes; excluir um pacote pode afetar
vários sandboxes.
