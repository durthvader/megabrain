# Pacotes compartilhados

Esta pasta contém somente código reutilizado por mais de um projeto. Ela não é
um projeto e não aparece no catálogo.

- `3ps/`: extrator regional e recursos visuais comuns dos painéis 3Ps.
- `horas/`: extração e geração das apresentações de Custo de Horas.

Excluir um sandbox não deve apagar estes pacotes; excluir um pacote pode afetar
vários sandboxes.
