# 3Ps Ceará

Sandbox canônico do painel e da apresentação executiva do Ceará.

- `src/extract_data.py` atualiza `data/processed/dados.json`.
- `src/build_presentation.py` gera o PowerPoint em `deliverables/`.
- `public/` é o pacote independente do painel; o PowerPoint publicado fica em `public/downloads/`.
- `docs/` contém briefing, análise e evidências específicas do Ceará.

Dados, código específico e entregáveis são privados e ignorados pelo repositório central.
