# 3Ps Maranhão / Piauí

Sandbox canônico do painel e da apresentação executiva de Maranhão/Piauí.

- `data/processed/dados.json` contém a fotografia usada pelo gerador.
- `src/build_presentation.py` gera o PowerPoint em `deliverables/`.
- `public/` é um snapshot publicável de forma independente.

Antes de publicar, reconcilie os números do HTML com o fechamento do PowerPoint.
